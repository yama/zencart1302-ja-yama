Zen Cart™のe-メールサポート

使い方 - リッチテキストエディタ / e-メールのアーカイビング / HTMLメールの送信

HTML/リッチテキストエディタのサポートを有効にする

エディターの設定はConfiguration | My Store |  HTML Editor で行うことができます。

リッチテキストでメッセージを作成する際の、e-メールシステムの設定方法

管理画面でConfiguration | E-Mail Optionsへ行ってください。

a. あなたの使っているWebサーバーに"e-メール配信方法"が正しく設定されているか確認してください。
b. メールを送るときはMIME HTMLを "true"にしてください
c. Send E-Mailsは "true"にしてください
d. E-mail アーカイビングはアクティブになっていますか？ この設定は"true"にしておくことをお勧めします。-- 注意　 この設定はデータベースサイズが恒常的に増大します。

HTMLメールを送るには、どこから行えばよいですか？

管理画面の"Tools" | "Newsletter Manager"からメルマガを配信することが可能です。.
この機能は商品通知機能と連動します。この商品通知はメルマガマネージャー画面から生成されます。

もしくは"Tools" | "Send Email"からも作成できます。もしくは管理画面の"Customers" | "Customers" で"e-メール"をクリックしてください。

クーポンメール, ギフトメールなども同様の方法で作成できます。

エディターを使わずに、e-メールに直接HTMLコードをコピーペーストしたらどうなりますか？

HTMLエディターの"ソース(source)"オプションをONにすることで、生HTMLを記述することができます。
さらに、エディターを無効にしたままでHTMLメール送信を有効にしておくと生HTMLを直接記述することができます。

コメント& FAQ

テキストのみ受信する設定にしているお客様に対してはどうなりますか？

HTMLメールを受け取れないメーラーを使っている人のために、下段に、テキストのみのコンテンツを入れる箇所があります。 "TEXTのみ"のエリアを空白にしておくと、HTMLコンテンツが変換されます。HTMLタグを削除するのみの処理になります。テキスト部のみが送られます。

HTMLメールの中のロゴを変更するには？

/email フォルダーの中に "header.jpg"というファイルがあります。.こちらを差し替えることでHTMLメールのロゴ画像を変更することができます。

メールの送信者のアドレスの見え方が変

"送信者"のメールアドレスに")"が入っている場合にそうなることがあります。

使っているブラウザでエディターが動作しない

お使いのブラウザでHTMLエディターが正しく動作しない場合、管理画面で、エディターを使用しない設定にしてください。そして、普段使っているHTMLエディターを用いて作成した生htmlを直接貼り付けてください。

バウンスのコントロールBOUNCE HANDLING

Zen Cartはメールバウンスのコントロールはできません。

メールがバウンスされた場合、管理画面の顧客（CUSTOMER）を選択し、検索ボックスにe-メールアドレスを入れて、該当者のメールアドレスを探し出し、該当者のアカウント設定を変更してください。

HTMLメールのテンプレートシステムの使い方

e-メールのテンプレートシステムは/email フォルダーの中にあるemail_*.htm を操作します。

e-メール テンプレートのスタイルシート

HTMLメールの標準形が広がり、全てのメーラー側で同じように見えるようにすることが不可能になりました。それゆえ、スタイルシートをリンクしておくことは推奨されません。いくつかのメーラーではスタイルシートのリンクを排除するものもあります。

HTMLメールのテンプレートを加工するにあたっては、HTML3.2とCSS1を用いて記述するのが最適です。
