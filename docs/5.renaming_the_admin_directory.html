管理者ディレクトリーのセキュリティー

この項目について

インストールを行った後に管理者（admin）ディレクトリーをリネームする追加の保護策を薦めます。このようにすることでハッカー達が管理者エリアを見つけたり攻撃を試みたりすることを著しく難しくします。

Admin Directoryのリネームの方法

変更を行う前にファイルとデータベースのバックアップを取りましょう。.

1- admin configurationを開きます。：　admin/includes/configure.php

このファイルの/admin/ の部分をあんたが選んだ名前に変更します。

この部分の変更：

// NOTE: be sure to leave the trailing '/' at the end of these lines if you make changes!
// * DIR_WS_* = Webserver directories (virtual/URL)
// these paths are relative to top of your webspace ... (ie: under the public_html or httpdocs folder)
define('DIR_WS_ADMIN', '/admin/');
define('DIR_WS_CATALOG', '/');
define('DIR_WS_HTTPS_ADMIN', '/admin/');
define('DIR_WS_HTTPS_CATALOG', '/');

この部分の変更：

// * DIR_FS_* = Filesystem directories (local/physical)
//the following path is a COMPLETE path to your Zen Cart files. eg: /var/www/vhost/accountname/public_html/store/
define('DIR_FS_ADMIN', '/home/mystore.com/www/public/admin/');
define('DIR_FS_CATALOG', '/home/mystore.com/www/public/');

2- admin directoryをリネームします。

FTPソフトを使って Zen Cart™ の/admin/ directoryを探します。そして、ステップ１でadmin/includes/configure.phpで編集した名前を使ってマッチするようにディレクトリーをリネームします。
